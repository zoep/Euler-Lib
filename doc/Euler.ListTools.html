Module Euler.ListTools

module ListTools: sig .. end
val map : ('a -> 'b) -> 'a list -> 'b list
Tail recursive version of map, slower but safe for large lists
val append : 'a list -> 'a list -> 'a list
Optimized append of two lists
val mklist : int -> int -> int list
mklist s f will create an int list containing [s;s+1;s+2...f]. Returns empty list if s > f. Tail recursive and fast.
val pairs : 'a list -> 'b list -> 'b list -> ('a * 'b) list -> ('a * 'b) list
pairs xs xs xs [] returns a list containing a tuple (x,y) with all possible combinations of elements of xs
